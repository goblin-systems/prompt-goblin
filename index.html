Get a Gemini API key

  1. Go to Google AI Studio (aistudio.google.com/apikey).
  2. Sign in, then click Create API key.
  3. Copy the key and paste it into the Gemini API key field.
  4. Click Test to confirm the key works.

Keep your API key private and never share it publicly.
